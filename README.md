# pickem-app
> Flask Application for the Confidence Pickem Endpoints

## Technologies
- Python3
- flask Blueprint
- werkzeug
- mysql-connector-python

## Getting Started
1. Install Python3
<br>
**MSI/ARM/TAR Installers**:<br>
https://www.python.org/downloads/
<br>
**Chocolately**:<br>
`choco install python3`
<br>
**Homebrew**:<br>
`brew install python3`
<br><br>

2. Serve the Application<br>
`python3 app.py`
